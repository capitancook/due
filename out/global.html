vDrawPolyline(vPoints)

vDrawPolyline is a viewport function used to draw a polyline passing truogh the points vPoints[0], vPoints[1], vPoints[vPoints.length-1]. Note that: 1) the polyline is automatically closed, this means that if vPoints[vPoints.length-1] <> vPoints[0] a line bteween these points is drawn to close the polyline; 2) the polylin is drawn only if there are more than two point in the vPoints array.
Parameters:
Name Type Description
vPoints Array.<Object> array of points in viewport coordinates.
